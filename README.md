## ğŸš€ Installation and Setup

### System Requirements

- Node.js (version 14.0.0 or higher)
- npm or yarn

### Installation Steps

1. **Clone Repository**
   ```sh
   git clone https://github.com/phucanhle/react-social.git
   cd react-social
   ```

2. **Install Dependencies**
   ```sh
   npm install
   # or
   yarn install
   ```

3. **Run Application**
   ```sh
   npm run dev
   # or
   yarn dev
   ```

Access the application at `http://localhost:5173`
